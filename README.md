# Shopping Application GUI

## Overview
The project is a Python GUI shopping app with an interactive chatbot to improve the user experience. The app is developed using the appJar library and enables the user to browse through sports merchandise in different categories such as jerseys, hats and accessories in a user friendly interface. Chatbot element receives the input of the user and responds by providing the user with the corresponding information of the user input such as product pricing through the usage of keywords and other responses will be learnt dynamically as the chatbot runs. The system is based on external data in a CSV file to provide the list of products and has incorporated some basic concepts of programming such as modular design, processing user input and basic data management. On the whole, the project has proven useful experience in the development of GUI, Python scripting, and development of interactive, data-driven applications.

## Features
- GUI built with appJar
- Category selection (Jerseys, Hats, Accessories)
- Separate chatbot logic file
- Reads product data from a CSV file

## How to Run
1. Install dependencies:
   pip install pandas appJar
2. Run the program:
   python codetemplate.py

## Files
- codetemplate.py: Main GUI application
- chatbot.py: Chatbot logic
- primethreads.csv: Product data
- PrimeThreads.gif: GUI image asset
